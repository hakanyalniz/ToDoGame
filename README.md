# To Do Quest

This is a web application that allows users to add skills to their status screen and practice those skills to increase their proficiency. This increased proficiency will result in experience gain and increase the skills level.

It is still ongoing.

## Goals

First, function over appearance:

- When enough experience is collected, skill's level will increase
- Create a proficiency system that calculates each click as experience, has internal experience rules and logic for each skill and levels up the skill when it is met.
